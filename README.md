# PoE2 Build Tracker

Lokalny notatnik do rozwijania postaci w Path of Exile 2.
Claude Code czyta te pliki i służy jako doradca buildu z pełnym kontekstem.

## Jak używać
W terminalu/Claude Code: wejdź do `c:\GitHUB\Poe2` i zapytaj o cokolwiek.
Pliki są aktualizowane na bieżąco razem z postępem gry.

## Struktura
- `builds/monk_invoker.md` — aktywny build: Monk Invoker
- `gear.md` — aktualne wyposażenie i cele itemowe
- `progress.md` — postęp w kampanii i endgame
- `notes.md` — obserwacje, pytania, eksperymenty
